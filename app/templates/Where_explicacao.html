Where

A função WHERE em SQL é usada para filtrar registros de uma tabela com base em condições especificadas. Ela determina quais linhas devem ser incluídas no resultado de uma consulta.

Por exemplo considere a tabela nomes

Nome Idade
João 25
Maria 30
Carlos 20

Eis a seguinte query :


        SELECT * 
        FROM tabela 
        WHERE idade >= 25;

Output

Nome Idade
João 25
Maria 30

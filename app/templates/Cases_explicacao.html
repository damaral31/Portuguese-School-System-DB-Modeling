CASE

A expressão CASE é usada para criar condições condicionais, permitindo retornar diferentes valores com base em critérios especificados. Funciona como uma estrutura if-else dentro de uma consulta.

Por exemplo considere a tabela nomes:

Nome Idade
Afonso 25
Miguel 30
Diogo 20

Eis a seguinte query :

SELECT nome
CASE
    WHEN idade > 25 THEN 'Velho'
    WHEN idade < 25 THEN 'Novo'
    ELSE 'Idade igual a 25'
END AS 'Classificação'

Output:

Vai ser criada uma nova coluna chamada Classificação que vai conter os valores 'Velho', 'Novo' ou 'Idade igual a 25' com base na idade de cada linha.

Nome Classificação
Afonso Idade igual a 25
Miguel Velho
Diogo Novo

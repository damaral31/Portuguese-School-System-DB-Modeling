Select

O comando SELECT em SQL é utilizado para selecionar linhas de uma tabela específica.
Por exemplo, considere a tabela nomes:

Nome Idade
Joana 18
Manuel 25
Carla 30



Eis a seguinte query: 

    
        SELECT * 
        FROM nomes;

Este comando irá selecionar todas as linhas da tabela nomes:

Nome Idade
Joana 18
Manuel 25
Carla 30


Para selecionar apenas algumas colunas, basta substituir o * pelo nome das colunas desejadas: 

    
        SELECT idade 
        FROM nome_da_tabela;

Esta query irá selecionar apenas a coluna idade da tabela nomes:
